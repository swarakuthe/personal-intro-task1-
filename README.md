# Personal Introduction Program

## Project Overview
This project is a simple Python program that collects personal and academic details from the user and displays a formatted profile summary.

## Features
- Takes user input (name, age, address, course, major, hobbies)
- Performs basic arithmetic (age next year)
- Uses lists to store favorite foods
- Displays formatted output

## Tools Used
- Python
- Google Colab
- GitHub

## How to Run
1. Open the `Task1.ipynb` file
2. Run the program using Python
3. Enter the required details when prompted
